# Agent Assets

Future War Room prompts, skills, and handoff templates live here.

- `skills/` is for local War Room-specific skills.
- `prompts/` is for reusable issue, PR, review, and recovery prompts.
- `plugins/marketplace.json` pins repo-local Codex plugins from `../plugins/` so handoff prompts and MCP entries do not depend on user-level plugin cache paths.

Do not commit secrets or provider credentials in this directory.
